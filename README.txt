# YandexTest Project README

## Описание
Проект предназначен для автоматизированного тестирования мобильного приложения Yandex.

## Необходимые инструменты
1. **Appium Server**
2. **Эмулятор Android**
3. **Файл APK (yandex.apk)**

## Порядок установки и запуска тестов

### 1. Установка Appium Server
Appium используется для автоматизированного тестирования мобильных приложений. Для установки Appium следуйте инструкциям на [официальном сайте Appium](http://appium.io/).
